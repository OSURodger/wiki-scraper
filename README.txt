This service product requires that python and some sevice packages to be installed.
First, download the lattest version of Python from the official website.
Next, into the command prompt, type "pip install requests" and
"pip install beautifulsoup4"", into the comand prompt

To use this in your program, at least for demonstration purposes,
you must open the python file itself, and change the input on line 33 from 'Joanne', 'Campbell'
to the desired 'firstName', 'lastName' of the actor you need data on.

Then, simply run the python code. The CSV will update with the bio info.
From there, it should be possible to read that CSV file's data into your program.
